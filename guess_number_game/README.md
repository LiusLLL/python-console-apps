# Guess Number Game

Консольная игра «Угадай число» на Python.

## Возможности
- Ввод никнейма игрока
- 3 уровня сложности:
  - Лёгкий: 1–50 (10 попыток)
  - Средний: 1–100 (7 попыток)
  - Сложный: 1–200 (5 попыток)
- Подсказки «больше / меньше»
- Команда `стоп` для выхода
- Сохранение результатов игры в файл `game_results.txt`

## Запуск
```bash
python guess_number.py
